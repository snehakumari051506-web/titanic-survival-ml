# Titanic Survival Prediction 🚢

A machine learning project predicting passenger survival on the Titanic using Logistic Regression.

## Results
- **Model:** Logistic Regression
- **Accuracy:** 81.08%
- **Dataset:** Kaggle Titanic Competition Dataset

## Features Used
- Pclass (Passenger class — 1st, 2nd, 3rd)
- Sex (converted to 0/1)
- Age (missing values filled with median)

## Libraries Used
- Python
- Pandas
- NumPy
- Scikit-learn

## What I Learned
- Data cleaning and handling missing values
- Feature engineering (converting categorical to numerical)
- Training and evaluating a classification model
- Understanding precision, accuracy and model evaluation
